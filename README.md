# arfvcr

Raspberry GPIO control for Trinicon

## Pinout

 - GPIO17 pin 11(in):  START/STOP
 - GPIO27 pin 13(out): TALLY SIG
 - GPIO22 pin 15(in):  REC/REVIEW
